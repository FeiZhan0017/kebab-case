# Tetris Clone

A minimal Tetris implementation using `pygame` for rendering and keyboard input.

## Requirements

1. Python 3.10+
2. Install dependencies:

   ```bash
   python -m pip install -r requirements.txt
   ```

## Running the Game

Launch the game with:

```bash
python -m src.main
```

## Controls

* `←` / `→`: Move the active piece left or right
* `↓`: Soft drop (faster descent)
* `↑` or `X`: Rotate clockwise
* `Z`: Rotate counter-clockwise
* `Space`: Hard drop
* `C`: Hold / swap current piece
* `Esc`: Quit the game
